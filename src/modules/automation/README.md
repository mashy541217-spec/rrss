# Automation Module

Coordina trabajos, ejecuciones, pasos, reintentos y politicas de automatizacion.

Responsabilidades futuras:

- programacion de trabajos;
- estados de ejecucion;
- reintentos;
- limites por negocio;
- bloqueo de duplicados;
- trazabilidad.

Este modulo decide que debe ejecutarse, pero no debe conocer detalles internos de cada plataforma.
