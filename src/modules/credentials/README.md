# Credentials Module

Gestiona metadatos y acceso seguro a credenciales sensibles.

Responsabilidades futuras:

- tokens de plataformas;
- credenciales de proxies;
- credenciales de dispositivos;
- referencias a secretos cifrados;
- rotacion y vencimiento.

Las credenciales nunca deben registrarse en logs ni exponerse a modulos que no las necesiten.
