# AI Agents Module

Gestiona capacidades de IA de forma controlada y auditable.

Responsabilidades futuras:

- adaptadores de OpenAI;
- adaptadores de Ollama;
- prompts versionados;
- politicas de agentes;
- ejecuciones de IA;
- limites de uso por negocio.

La IA debe asistir workflows, no operar sin control ni trazabilidad.
