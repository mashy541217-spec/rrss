# Businesses Module

Gestiona los negocios independientes dentro de RRSS AUTO.

Responsabilidades futuras:

- perfil del negocio;
- configuracion por negocio;
- zona horaria;
- idioma;
- voz de marca;
- plataformas habilitadas;
- limites operativos.

Este modulo es central porque todo dato operativo debe estar asociado a un negocio.
