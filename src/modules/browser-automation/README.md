# Browser Automation Module

Gestiona automatizacion basada en navegador.

Responsabilidades futuras:

- perfiles de navegador;
- sesiones;
- Playwright;
- capturas;
- trazas;
- asignacion de proxy;
- recuperacion ante fallos.

La automatizacion de navegador es fragil por naturaleza, por eso debe estar aislada y muy observada.
