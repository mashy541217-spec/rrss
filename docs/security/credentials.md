# Credenciales

Las credenciales son activos criticos.

## Reglas

- No se almacenan secretos en repositorio.
- No se imprimen secretos en logs.
- No se comparten credenciales entre negocios.
- Toda credencial debe tener propietario, proposito y fecha de rotacion cuando aplique.

La implementacion concreta se definira en un sprint posterior.
