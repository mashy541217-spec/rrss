# Automation Strategy

La automatizacion debe ser observable, controlada y recuperable.

La base arquitectonica de toda automatizacion esta definida en `docs/rfc/RFC-0001-execution-engine.md`.

Tipos previstos:

- APIs oficiales;
- Playwright;
- Android;
- VMware;
- tareas asincronas;
- workflows asistidos por IA.

Toda automatizacion debe registrar trabajo, ejecucion, pasos, resultado y artefactos relevantes.
