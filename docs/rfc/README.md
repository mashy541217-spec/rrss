# RFCs

Esta carpeta contiene Request for Comments arquitectonicos de RRSS AUTO.

Un RFC describe una propuesta fundacional o un subsistema completo antes de su implementacion. No reemplaza a una ADR: el RFC explica el diseno amplio, mientras que las ADRs registran decisiones especificas aceptadas.

## RFCs actuales

- `RFC-0001-execution-engine.md`: modelo de ejecucion central de RRSS AUTO.
